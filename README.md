# 🌤 WeatherInsight Pro

WeatherInsight Pro is an advanced weather dashboard built using Python and Streamlit. The application provides real-time weather information, air quality insights, forecast analytics, favorite city management, and city-to-city weather comparison.

## 🚀 Live Demo

https://synent-task6-weatherapp-jafar-jrjsqt6x87zhmgcf3kxwwx.streamlit.app/

## 📂 GitHub Repository

https://github.com/JAFAR-666/synent-task6-weatherapp-jafar

## ✨ Features

- Real-time Weather Information
- Temperature, Humidity, Wind Speed
- Feels Like Temperature
- Pressure & Visibility
- Weather Condition Icons
- Air Quality Index (AQI)
- Favorite Cities
- Compare Cities
- Forecast Analytics
- Interactive Charts using Plotly
- Responsive Streamlit Interface

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Plotly
- Pandas
- OpenWeatherMap API

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/JAFAR-666/synent-task6-weatherapp-jafar.git
cd synent-task6-weatherapp-jafar
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
