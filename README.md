# WeatherInsight Pro

Advanced Weather Dashboard built using Python and Streamlit.

## Features

- Real-time Weather Information
- Temperature, Humidity, Wind Speed
- Air Quality Index (AQI)
- Favorite Cities
- City Comparison
- Forecast Analytics
- Interactive Charts

## Technologies Used

- Python
- Streamlit
- Requests
- Plotly
- OpenWeatherMap API

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
